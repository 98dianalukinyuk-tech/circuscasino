# Circus — Pokerroom & Toernooigids

- Taal/locale: nl-BE
- Geo: België
- Valuta: EUR
- Tijdzone: Europe/Amsterdam
- Pagina's: 17

## Lokaal openen
Open `index.html` rechtstreeks, of start in deze map `python -m http.server 8080` en bezoek `http://localhost:8080/`.

## Voor publicatie
Vervang alle `{{VERIFIED_*}}`-waarden in `script.js`, canonical/OG-velden en `sitemap.xml` door gecontroleerde gegevens. Verwijder `noindex,nofollow` en wijzig `robots.txt` pas na juridische, merk- en contentcontrole.

Dit project gebruikt een origineel gidslogo en geen officieel merklogo. De bezoekersgids vermeldt dat zij niet door de merkhouder wordt beheerd.
